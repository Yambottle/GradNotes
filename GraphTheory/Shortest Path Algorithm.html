1. Shortest Path Algorithm:->does not support negative cycle
    1. Dijkstra’s=> shortest path from one node to all other nodes
      1. does not support negative weighted edge
      2. find closest available vertex to update shortest path
      3. Optimize:
        1. adjacency matrix
        2. heap sort
        3. Fibonacci sort(heap)
          1. Heap
            1. Binary Heap
            2. Binomial Heap
              1. Binomial Tree: set of trees
              2. Degree: power of 2, how many children which root has
              3. recursive to build a binomial tree(combine each Ndegree-1 binomial tree)
              4. Binomial Heap is a collection(forest) of binomial trees
              5. Consolidate: doesn’t allow same degree binomial tree
            3. Fibonacci Heap:lazy version of binomial heap
              1. allow same degree binomial tree while insertions or merging
              2. only consolidate while extractions
              3. use circle doubled linked list to store all roots and all children at the same level
    2. Bellman-Ford=>literately find shortest path for nodes (S A B C D)
    3. Floyd-Warshall=>all pair shortest path
