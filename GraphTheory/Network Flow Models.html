1. General Problem: Minimum Cost Flow Problem
    1. Define a network G(N,A)
    2. for each node i in N, b(i) is an associated value
      1. if b(i)<0 demand node
      2. if b(i)==0 transparent node
      3. if b(i)>0 supply node
    3. xij: units of flows on arc
    4. cij: cost
    5. uij:upper bond of flow, capacity of arc
    6. lij: lower bond, usually 0
  2. Use case 1: Shortest Path Problem (weight=cost): apply Dijkstra (No negative weight)
  3. Use case 2: Max Network Flow Problem
