1. Network flow problem: a type of network optimization problem
    1. Arise as many packages/trucks(Flow) as possible
  2. How to solve?=> LP is one of ways.
  3. Now the flow(decision variable) could be fraction value, if want integer, then it’s a integer programming problem
  4. What’s the foundational Theory?
    1. Menger ’s Theorem (general)
      1. cut vertex in a graph
      2. How many vertices need to be remove if we want two vertices separate?
        1. Minimum separating vertex set
      3. Internal disjoint path: P1: v1 v2 v3 v4; P2: v1 v5 v6 v4;
    2. The cardinality of minimum separating vertex set of vertex u and v is equal to the maximum number of internal disjoint paths between u and v
    3. Find a minimum cut between Source and Sink=> Find a maximum flow path from Source to Sink
      1. # of edges/ Product of endpoints
    4. For any 2 vertices u and v in graph G, a u-v separating set is a set of vertices whose deletion from G results in a disconnected graph in which u and v are in distinct components
    5. A minimum u-v separating set is a u-v separating set of minimum cardinality on size(# of vertices)
    6. A set of u-v paths is called internally disjoint if every two paths have no vertices in common other than u and v
    7. Menger ’s Theorem:
      1.  The cardinality of minimum separating vertex set of vertex u and v is equal to the maximum number of internal disjoint paths between u and v 
  5. Flow:
    1. Integer Flow definition: flow-in and flow-out on edges are integers
    2. A k-flow is an integer flow such that every edge is assigned a weight whose absolute value is strictly less than k
      1. k-flow: the largest absolute flow weight value +1 (for increase in the future LP problem)
    3. A nowhere-zero flow is a flow in which no edge is assigned a weight of zero
    4. bridges can only have 0 flow, because there are imbalanced part aside
      1. bridge is also called cut-edge
    5. A bond is a minimal edge cut(minimal edge set)
      1. edge cut is a set of edges which can separate the graph after deleting those edges
    6. Theorem: the set flow across any bond is Zero
    7. Three Theorems:
      1. Every connected bridge-less graph admits a nowhere-zero=>8-flow
      2. 6-flow
      3. Conjunctive but should be correct
    8. Face Coloring Setting Flow
      1. randomly and manually set face color
      2. Flow direction following Right Hand Law: right color number < left color number
      3. Flow weight: Difference between right and left color number
