  10. Diameter/Radius:
    1. eccentricity: max length of point to all other points
    2. diameter: maximum length of shortest path between any pair of vertices
    3. radius: minimum ….
    4. if a vertex V’s eccentricity = diameter, called peripheral vertex of G
      1. set of peripheral vertices called periphery
    5. if a vertex V’s eccentricity = radius, called central vertex
      1. set called centre
    6. radius<=diameter<=2*radius
    7. if G is a disconnected graph, then G’ is connected and diameter(G’)<=2
